Syntax themes for <a href="http://akelpad.sf.net/">AkelPad</a> text editor with <a href="http://akelpad.sf.net/en/plugins.php">Coder plugin</a>.
<br>Note: <strong>contains many non-standard variables</strong>, see <a href="_colors.html">\_colors.html</a> or <a href="_colors.txt">\_colors.txt</a> for reference.
<br>
#### Simplified installation using pre-configured <a href="Coder.ini">Coder.ini</a>:
* replace `VarThemeGlobal=…` in your `Coder.ini`, if not used global variables;
* or copy new parameter `/Infocatcher=…` to your `Coder.ini` to add syntax theme (icons: <a href="https://github.com/Infocatcher/AkelPad_menu/tree/master/icons">Infocatcher/AkelPad_menu/icons</a>).

Also you can use <a href="https://akelpad.sourceforge.net/forum/viewtopic.php?p=17303#p17303">PluginText.js</a> script (with <a href="https://akelpad.sourceforge.net/forum/viewtopic.php?t=202">Scripts plugin</a>) to edit Coder theme.
<br>
<br>And some syntax themes are just a bit modified ones from other authors.
<br>
<br>All files are prefixed to avoid overrides after update.
